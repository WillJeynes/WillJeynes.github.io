Accessibility

Accessibility Statement

This website is committed to providing a brilliant user experience for all people, irrespective of ability or access to technology. This website works in all modern web browsers, and has taken steps in order to make the website content clear to screen readers, and keyboard navigators.

Keeping unnecessary multimedia elements away from this website was a priority, meaning no videos, or unnecessary background images have been used in this website.

For any enquiries, or to report any problems using the website, contact me here.

Reasoning for choices

I feel it is important to make this website accessible to all, therefore technologies such as image alternative text and clear navigation were important to me. Because of the use of HTML5 in this website, browser extension based screen readers should be able to quickly and easily parse the website content, improving usability (Mozilla's guide of HTML semantics for accessibility)

Using some of the steps outlined in the Web Content Accessibility Guidelines this website has been created to be accessible to as many people as possible.

Some of the key points include:

  • Pages have been built with the answers in plain text, meaning those with screen readers could still understand the content.
  • Alternative text has been provided for all images in the website, in order to improve navigation for the visually impaired.
  • All text and images in this website feature clear contrasting colors, in order to improve navigation for all.
  • All content in this website features a larger font typeface and main website links have a large selectable area, in order to simplify navigation.
